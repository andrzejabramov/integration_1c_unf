    📋 Документация: Интерфейсы и объекты учета в 1С:УНФ

    🏭 Производственное предприятие

    📌 Как пользоваться документом

    Элемент Значение
    📘 Справочник Базы данных (контрагенты, товары и т.д.)
    📄 Документ Операции, которые вы создаете
    📊 Отчет Аналитика и сводные данные
    ⚙️ Настройки Конфигурация системы

    1️⃣ Отдел продаж (Менеджеры по продажам)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Контрагенты Главное → Cправочники → Все справочники → CRM → Контрагенты ✅ Юр. лицо, ИНН, договор по умолчанию
    📘 Номенклатура Главное → Cправочники → Все справочники → Общее → Номенклатура ⚠️ Только товары/услуги, которые продаем (вид "Товар")
    📘 Коммерческие предложения Отсутствуют 📝 Шаблон для быстрой печати КП клиенту
    📄 Заказ клиента Продажи и Сервис → Заказы клиентов 🎯 Основа для работы! Заполнять: контрагент, номенклатура, цены, сроки
    📄 Реализация товаров и услуг Продажи → Заказы Покупателей 📦 Оформляется после отгрузки (или на основе Заказа)

    2️⃣ Главный инженер (Технический директор)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Основные средства Компания → Справочники → План счетов управленческого учета 🏭 Оборудование цехов, здания, дорогой инвентарь
    📘 Виды работ (операции) Производство → Задание на работу → Виды работ 🔨 Список операций (Сварка, Фрезеровка и т.д.), расценки
    📄 График ремонтов (ППР) Отсутствует 📅 Плановое обслуживание оборудования
    📄 Заказ-наряд на ремонт Отсутствует 🔧 Фиксация факта поломки, затрат времени и материалов

    3️⃣ 📊 Инженер ПДО (Планово-диспетчерский отдел)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Спецификации (Состав изделия) Производство → Справочники → Спецификация 🧩 Что во что входит. Уровни вложенности, нормы расхода
    📘 Маршрутные карты (Техкарты) Производство → Отчеты → Нормативный состав изделия ⏱️ Последовательность операций изготовления
    📄 Заказ на производство Производство → Заказы на производство 🚀 Запуск изделия в работу. Заполнять: номенклатура, количество, плановая дата
    📄 Сменное задание Производство → Задания и наряды → Задания на работу 👷 Постановка задач на смену по участкам/рабочим центрам

    4️⃣ ⚙️ Технолог

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Номенклатура Производство → Справочники → Номенклатура 🏭 Создание готовой продукции, полуфабрикатов. Тип: "Товар"
    📘 Спецификации (Состав изделия) Производство → Справочники → Спецификация 📦 Материальный состав (что входит). Вид: "Сборочный узел"
    📘 Технологические карты (Техкарты) Производство → Справочники → Спецификация 🛠️ Описание операций, переходов, инструмента, трудозатрат
    📘 Инструменты и оснастка Склад → Номенклатура 🔪 Режущий инструмент, штампы, приспособления

    5️⃣ 📐 Конструктор

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Номенклатура Производство → Справочники → Номенклатура 📝 Создание позиций чертежей, деталей (на стадии разработки)
    📘 Спецификации Производство → Справочники → Спецификация 🏗️ Конструкторский состав изделия (вложенность)
    📎 Внешние файлы (вложения) Кнопка "📎 Скрепка" в любом элементе 📁 Прикрепление чертежей PDF, DXF, 3D-моделей к номенклатуре или спецификации

    6️⃣ Снабженец (Отдел закупок)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Поставщики Закупки → Поставщики 🤝 Основные поставщики, условия оплаты, сроки отгрузки
    📄 Заказ поставщику Закупки → Заказы поставщикам 📋 Формирование потребности. Заполнять: товар, кол-во, срок, цена
    📄 Поступление товаров и услуг Закупки → Приходные накладные 📥 Оприходование ТМЦ на склад (на основе Заказа поставщику)
    📊 Потребность в закупках Отсутствует 🔍 Анализ: что нужно купить по необеспеченным заказам

    7️⃣ Кладовщик (Материалы и НЗП)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Склады (ячейки) Склад → Склады и магазины 🏢 Зона "Материальный склад", зона "Цех (НЗП)"
    📄 Оприходование товаров Склад → Оприходование запасов (или Акт оприходования) ✅ Приемка материалов от снабжения
    📄 Перемещение товаров] Склад → Перемещения запасов 🚚 Выдача материалов в цех (со склада в НЗП)
    📄 Инвентаризация товаров Склад → Инвентаризации 🔢 Сверка остатков на складе материалов
    📊 Остатки товаров Склад → Аналитика → Отчеты → Остатки товаров 👀 Проверка наличия материала

    8️⃣ 🏭 Начальник производства (Мастер цеха)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Рабочие центры (Станки/Участки) Производство → Справочники → Бригвды 🔧 Станки, верстаки, участки (Фрезерный, Сварочный)
    📄 Сменное задание Производство → Задания и наряды → Задания на работу 📅 План для рабочих на день
    📄 Отчет производства за смену Производство → Аналитика → Отчеты → Выпуск продукции ✅ Фиксация выработки: сколько сделали, сколько времени потратили
    📄 Передача в производство Производство → Заказы на производство 📤 Списание материалов в работу (выдача со склада в НЗП)

    9️⃣ Кладовщик (Готовая продукция — ГП)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Склады (ГП) Склад → Склады и магазины (готовой продукции) Склад "Готовая продукция"
    📄 Оприходование товаров (Выпуск) Склад → Оприходование запасов (на склад брака) 📥 Приемка ГП из цеха на склад (на основе Отчета за смену)
    📄 Отгрузка (Реализация) Склад → Списание 🚚 Отгрузка со склада ГП клиенту
    📄 Комплектация (Сборка) Склад → Комплектация запасов 📦 Сборка набора изделий или упаковка
    📊 Ведомость по складу Склад → Аналитика → Отчеты → Остатки товаров 📋 Остатки ГП в разрезе склада

    🔟 ✅ Инженер ОТК (Отдел технического контроля)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📄 Отчет производства за смену (контроль) Производство → Аналитика → Отчеты → Выпуск продукции ⚠️ Вход в документ, статус "Брак" / "Годная". Заполнение параметров качества
    📄 Акт о браке Производство → Заказ на производство → Перемещение (на склад брака) ❌ Оформление исправимого/неисправимого брака. Виновник, статья затрат
    📄 Приемка ТМЦ (Качество) Склад → Оприходование запасов (на склад брака) ↩️ Оформление возврата поставщику при выявлении несоответствия материалов

    1️⃣1️⃣ Отдел реализации (Склад ГП + Логистика/Отгрузка)

    💡 Примечание: Часто функции делятся между менеджером по продажам и кладовщиком ГП. Здесь объединены складские функции отгрузки.

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📄 Заказ клиента Продажи → Заказы покупателей 🔒 Бронирование ГП под отгрузку
    📄 Реализация товаров и услуг Продажи → Аналитика → Отчеты → Продажи 📤 Оформление отгрузки, ТОРГ-12
    📄 Счет на оплату Продажи → Счета → Счет на оплату 💰 Выставляется клиенту перед отгрузкой
    📄 Отчет комиссионера Есть комиссионная торговля 🤝 Продажа через посредника

    📘 1С:УНФ — Экономист

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Статьи затрат Компания → Отчеты → Пульс бизнеса 📋 Классификатор расходов (Материалы, Зарплата, Амортизация, Накладные расходы). ⚠️ Важно для себестоимости!
    📘 Статьи движения ДС Деньги → Справочники → Статьи движения денег 💰 Платежные статьи для Платежного календаря (Оплата поставщикам, Налоги, ФОТ)
    📘 Номенклатурные группы Главное → Справочники → Номенклатура (через создать) 📦 Группировка продукции (например, "Корпусная мебель") для анализа рентабельности
    📘 Проекты / Подразделения Отсутствуют 🎯 Разрез аналитики для детализации затрат (по цехам, заказам, направлениям)
    📄 Платежное поручение Финансы → Платежные поручения (исходящие) 📤 Планирование и отражение оплаты поставщикам, налогов, зарплаты
    📄 Платежный календарь Отсутствует 📅 Управление лимитами и планирование платежей на день/неделю
    📄 Заявка на расходование ДС Отсутствует 📝 Внутренняя заявка от подразделений на оплату (снабжение, производство)
    📄 Списание с расчетного счета Деньги → Банк → Расходы со счета 🏦 Фактическое списание ДС (загрузка из банк-клиента)
    📄 Поступление на расчетный счет Деньги → Банк → Поступление на счет 📥 Загрузка фактических поступлений от клиентов
    📄 Корректировка долга Деньги → Аналитика → Отчеты ⚖️ Для закрытия взаимозачетов, списания задолженности
    📄 Распределение доходов и расходов Деньги → Аналитика → Отчеты 🔄 Распределение накладных расходов по направлениям (закрытие месяца)
    📊 Финансовый результат Компания → Аналитика → Отчеты → Финансовый результат 🎯 Ключевой отчет! Выручка, себестоимость, прибыль в разрезе групп, заказов, проектов
    📊 ОСВ (управленческая) Главное → Аналитика → Отчеты → Оборотно-сальдовая ведомость 📈 Обороты по счетам управленческого учета
    📊 Задолженность контрагентов Деньги→ Аналитика → Отчеты → Задолженность покупателей ⚠️ Контроль дебиторской и кредиторской задолженности
    📊 Движение денежных средств (ДДС) Деньги → Аналитика → Отчеты → Движение денежных средств 💵 Анализ платежей и поступлений по статьям ДС
    📊 Себестоимость продукции Производство → Отчеты → Себестоимость продукции 🧮 Калькуляция себестоимости единицы продукции (материалы, работа, накладные)
    📊 План-фактный анализ Компания → Аналитика → Состояние компании → Настройки рассылки 📊 Сравнение плановых показателей с фактическими

    ➕ Дополнительные объекты (если включены расширенные функции)

    Тип Название объекта Путь в интерфейсе 💡 Примечание
    📄 Бюджет (Сценарий) Финансы → Бюджетирование → Бюджеты 📋 Плановые показатели доходов, расходов, ДДС на период
    ⚙️ Установка лимитов расходов Финансы → Лимиты расходов ДС 🔒 Контроль расходов подразделений в рамках бюджета
    📘 Касса (операционная) НСИ и Администрирование → НСИ → Кассы 💵 Для учета наличных (если экономист контролирует кассу)

    1️⃣2️⃣ 👔 Руководитель

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📊 Финансовый результат Компания → Аналитика → Отчеты → Анализ бизнеса 💵 Валовая прибыль: Доходы, расходы, прибыль
    📊 Задолженность контрагентов Компания → Аналитика → Состояние компании ⚠️ Контроль дебиторки/кредиторки
    📊 Производственный анализ Производство → Аналитика → Отчеты 📈 Контроль затрат
    🖥️ Монитор заказов Продажи → Аналитика → Отчеты ⚡ "Живая" лента статусов заказов клиентов
    ⚙️ Установка лимитов расходов Деньги → Справочники → Статьи движения денег 💰 Бюджетирование, контроль оплат

    🎯 Ключевые рекомендации

    ✅ Для всех сотрудников:

    • 💾 Регулярно сохраняйте документы
    • 🔍 Проверяйте заполнение обязательных полей (помечены ⚠️)
    • 📅 Соблюдайте последовательность документов
    • 💬 Связывайтесь с коллегами при возникновении вопросов

    🔥 Важные моменты:

    1. Заказ клиента — основа работы отдела продаж
    2. Спецификации — сердце производственного планирования
    3. Отчет производства за смену — главный документ учета выработки
    4. Реализация — финальный документ отгрузки

    📝 Документ подготовлен для эффективной работы в 1С:УНФ. При возникновении вопросов обращайтесь к администратору системы!
