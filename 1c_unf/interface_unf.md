# 📋 Документация: Интерфейсы и объекты учета в 1С:УНФ
## 🏭 Производственное предприятие

---

## 📌 Как пользоваться документом

| Элемент | Значение |
|---------|----------|
| 📘 **Справочник** | Базы данных (контрагенты, товары и т.д.) |
| 📄 **Документ** | Операции, которые вы создаете |
| 📊 **Отчет** | Аналитика и сводные данные |
| ⚙️ **Настройки** | Конфигурация системы |

## 1️⃣  Отдел продаж (Менеджеры по продажам)

| Тип | Название объекта | Путь в интерфейсе | 💡 Основные поля / Примечание |
|:---:|------------------|-------------------|-------------------------------|
| 📘 | **Контрагенты** | `НСИ и Администрирование` → `НСИ` → `Контрагенты` | ✅ **Юр. лицо**, **ИНН**, договор по умолчанию |
| 📘 | **Номенклатура** | `НСИ и Администрирование` → `НСИ` → `Номенклатура` | ⚠️ Только товары/услуги, которые продаем (**вид "Товар"**) |
| 📘 | **Коммерческие предложения** | `CRM и Маркетинг` → `Коммерческие предложения` | 📝 Шаблон для быстрой печати КП клиенту |
| 📄 | **Заказ клиента** | `Продажи и Сервис` → `Заказы клиентов` | 🎯 **Основа для работы!** Заполнять: контрагент, номенклатура, цены, сроки |
| 📄 | **Реализация товаров и услуг** | `Продажи и Сервис` → `Реализации` | 📦 Оформляется **после отгрузки** (или на основе Заказа) |

| 📄 | **Заказ-наряд на ремонт** | `Склад и Доставка` → `Техобслуживание` → `Заказ-наряды` | 🔧 Фиксация факта поломки, затрат времени и материалов |

## 1️⃣1️⃣  Отдел реализации (Склад ГП + Логистика/Отгрузка)

> 💡 **Примечание:** Часто функции делятся между менеджером по продажам и кладовщиком ГП. Здесь объединены складские функции отгрузки.

| Тип | Название объекта | Путь в интерфейсе | 💡 Основные поля / Примечание |
|:---:|------------------|-------------------|-------------------------------|

| 📄 | **Реализация товаров и услуг** | `Продажи и Сервис` → `Реализации` | 📤 Оформление отгрузки, **ТОРГ-12** |
| 📄 | **Счет на оплату** | `Продажи и Сервис` → `Счета` | 💰 Выставляется клиенту **перед отгрузкой** |
| 📄 | **Отчет комиссионера** | Если есть комиссионная торговля | 🤝 Продажа через посредника |

## 🎯 Ключевые рекомендации

### ✅ Для всех сотрудников:
- 💾 **Регулярно сохраняйте** документы
- 🔍 **Проверяйте заполнение** обязательных полей (помечены ⚠️)
- 📅 **Соблюдайте последовательность** документов
- 💬 **Связывайтесь с коллегами** при возникновении вопросов

### 🔥 Важные моменты:
1. **Заказ клиента** — основа работы отдела продаж
2. **Спецификации** — сердце производственного планирования
3. **Отчет производства за смену** — главный документ учета выработки
4. **Реализация** — финальный документ отгрузки
