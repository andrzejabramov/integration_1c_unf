---

## 📌 Как пользоваться документом

| Элемент | Значение |
|---------|----------|
| 📘 **Справочник** | Базы данных (контрагенты, товары и т.д.) |
| 📄 **Документ** | Операции, которые вы создаете |
| 📊 **Отчет** | Аналитика и сводные данные |
| ⚙️ **Настройки** | Конфигурация системы |

---

## ⚠️ Важное отличие КА от УНФ

> 💡 **КА построена на принципе разделения учета:**
> - 📊 **Управленческий учет** — для внутреннего использования
> - 📑 **Регламентированный учет** — для бухгалтерии и налоговой
> 
> Для многих операций существуют **два документа** — для управленческого и для бухгалтерского учета!

## 🎯 Рекомендации для экономиста

- 🔍 **Проверяйте аналитику** при вводе документов (ЦФО, проекты, номенклатурные группы)
- 💾 **Регулярно проводите закрытие месяца** для корректного расчета себестоимости
- 📊 **Используйте план-фактный анализ** для контроля отклонений
- 🤝 **Согласовывайте заявки на ДС** с руководителями подразделений

### 🔥 Важные моменты в КА:
1. **Разделяйте управленческий и регламентированный учет** — не путайте документы!
2. **Используйте ЦФО** для детализации ответственности по затратам
3. **Настраивайте бюджетные правила** — это экономит время при планировании

### ⚠️ Частые ошибки:
| Ошибка | Последствие | Как избежать |

| ❌ Не заполнены аналитики в статьях затрат | Некорректная себестоимость | ✅ Проверять обязательные разрезы аналитики |
| ❌ Пропущено распределение накладных расходов | Заниженная/завышенная прибыль | ✅ Запускать регламентные операции закрытия месяца |
| ❌ Не утверждены заявки на ДС | Блокировка платежей в календаре | ✅ Своевременно согласовывать заявки |
