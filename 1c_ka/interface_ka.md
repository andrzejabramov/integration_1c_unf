---

## 📌 Как пользоваться документом

| Элемент | Значение |
|---------|----------|
| 📘 **Справочник** | Базы данных (контрагенты, товары и т.д.) |
| 📄 **Документ** | Операции, которые вы создаете |
| 📊 **Отчет** | Аналитика и сводные данные |
| ⚙️ **Настройки** | Конфигурация системы |

---

## ⚠️ Важное отличие КА от УНФ

> 💡 **КА построена на принципе разделения учета:**
> - 📊 **Управленческий учет** — для внутреннего использования
> - 📑 **Регламентированный учет** — для бухгалтерии и налоговой
> 
> Для многих операций существуют **два документа** — для управленческого и для бухгалтерского учета!

## 📘 1С:Комплексная автоматизация (КА) — Экономист

> 💡 **В КА функционал экономиста значительно шире** за счет разделения управленческого и регламентированного учета, наличия полноценного блока бюджетирования и более развитого учета затрат.

| Тип | Название объекта | Путь в интерфейсе | 💡 Основные поля / Примечание |
|:---:|------------------|-------------------|-------------------------------|
| 📘 | **Статьи затрат** | `Финансы` → `Настройки и справочники` → `Статьи затрат` | 📋 Управленческие статьи. ⚠️ **Есть иерархия и правила распределения** |
| 📘 | **Статьи движения ДС** | `Финансы` → `Настройки и справочники` → `Статьи движения ДС` | 💰 **Ключевой справочник** для Платежного календаря и анализа ДДС |
| 📘 | **Номенклатурные группы** | `НСИ и Администрирование` → `НСИ` → `Номенклатурные группы` | 📦 Для аналитики **рентабельности по группам продукции** |
| 📘 | **Подразделения (ЦФО)** | `НСИ и Администрирование` → `НСИ` → `Подразделения` | 🎯 В КА подразделения могут быть **Центрами финансовой ответственности** |
| 📘 | **Проекты** | `НСИ и Администрирование` → `НСИ` → `Проекты` | 📁 Дополнительный разрез аналитики (заказы, гранты, направления) |
| 📄 | **Заявка на расходование ДС** | `Казначейство` → `Заявки на расходование ДС` | 📝 Внутренний документ-основание. ⚠️ **Утверждается экономистом/финдиректором** |
| 📄 | **Платежный календарь** | `Казначейство` → `Платежный календарь` | 📅 Планирование платежей с учетом **лимитов и приоритетности** |
| 📄 | **Платежное поручение** | `Казначейство` → `Платежные поручения` | 📤 Создаются на основании **утвержденных заявок** |
| 📄 | **Выписка банка (загрузка)** | `Казначейство` → `Выписки банка` | 🏦 Загрузка фактических движений ДС, **автоматическое отражение** в учете |
| 📄 | **Распределение доходов и расходов** | `Финансы` → `Распределение доходов и расходов` | 🔄 **Закрытие месяца**, распределение накладных расходов по ЦФО, проектам |
| 📄 | **Расчет себестоимости** | `Регламентированный учет` → `Закрытие периода` → `Расчет себестоимости` | ⚙️ **Регламентная операция** для расчета фактической себестоимости выпуска |
| 📄 | **Корректировка долга** | `Финансы` → `Корректировка долга` | ⚖️ Для списания задолженности, взаимозачетов |
| 📄 | **Бюджет (Сценарий)** | `Бюджетирование` → `Бюджеты` | 📊 Плановые показатели по доходам, расходам, ДДС, балансу. **Поддержка сценариев: План, Факт, Прогноз** |
| ⚙️ | **Бюджетное правило** | `Бюджетирование` → `Настройки` → `Бюджетные правила` | 🤖 Правила **автоматического расчета** бюджетов на основе данных прошлых периодов |
| 📊 | **Финансовый результат (упр.)** | `Финансы` → `Отчеты` → `Финансовый результат` | 🎯 Анализ прибыли и рентабельности в разрезе групп, ЦФО, проектов |
| 📊 | **ОСВ (управленческая)** | `Финансы` → `Отчеты` → `Оборотно-сальдовая ведомость` | 📈 Управленческие обороты по счетам |
| 📊 | **Задолженность контрагентов** | `Финансы` → `Отчеты` → `Задолженность контрагентов` | ⚠️ Дебиторка и кредиторка в **управленческом учете** |
| 📊 | **Отчет о движении ДС** | `Казначейство` → `Отчеты` → `Отчет о движении ДС` | 💵 **План-фактный анализ** платежей по статьям ДС |
| 📊 | **Себестоимость продукции** | `Производство` → `Отчеты` → `Себестоимость продукции` | 🧮 **Детальная калькуляция** себестоимости (материалы, работа, косвенные расходы) |
| 📊 | **План-фактный анализ бюджета** | `Бюджетирование` → `Отчеты` → `План-фактный анализ` | 📊 Сравнение плановых и фактических показателей по бюджетам |
| 📊 | **Анализ затрат** | `Производство` → `Отчеты` → `Анализ затрат` | 🔍 Детализация затрат по статьям, заказам, подразделениям |

## 🔑 Ключевые отличия: Экономист в УНФ vs КА

| Аспект | 📘 1С:УНФ | 📘 1С:Комплексная автоматизация |
|--------|-----------|--------------------------------|
| **Учет** | Преимущественно управленческий. Регламентированный — упрощенный | ✅ **Полноценное разделение**: управленческий + регламентированный в единой базе |
| **Бюджетирование** | Базовое, ограниченное количество сценариев | 🚀 **Развитое**: множественные сценарии, бюджетные правила, консолидация, план-факт |
| **Казначейство** | Заявки, платежный календарь, лимиты | 🏦 **Полноценный блок "Казначейство"**: маршрутизация заявок, лимитирование, интеграция с банками |
| **Себестоимость** | Калькуляция, распределение накладных расходов | 🧮 **Сложный алгоритм**: попередельный метод, РАУЗ, расширенная аналитика затрат |
| **Закрытие месяца** | Упрощенная процедура | ⚙️ **Регламентная процедура** с четкой последовательностью шагов |
| **Регламентированная отчетность** | Не является основной целью | 📑 **Полный набор** отчетов для ФНС, ПФР, ФСС, Росстата |
