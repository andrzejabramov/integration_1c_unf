    📋 Документация: Интерфейсы и объекты учета в 1С:Комплексная автоматизация (КА)

    🏭 Производственное предприятие

    📌 Как пользоваться документом

    Элемент Значение
    📘 Справочник Базы данных (контрагенты, товары и т.д.)
    📄 Документ Операции, которые вы создаете
    📊 Отчет Аналитика и сводные данные
    ⚙️ Настройки Конфигурация системы

    ⚠️ Важное отличие КА от УНФ

    💡 КА построена на принципе разделения учета: - 📊 Управленческий учет — для внутреннего использования - 📑 Регламентированный учет — для бухгалтерии и налоговой

    Для многих операций существуют два документа — для управленческого и для бухгалтерского учета!

    1️⃣ Отдел продаж (Менеджеры по продажам)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Контрагенты НСИ и Администрирование → НСИ → Партнеры ✅ Юр. лицо, ИНН. ⚠️ Важно: заполнить категорию (Покупатель, Поставщик)
    📘 Номенклатура НСИ и Администрирование → НСИ → Номенклатура 📦 Вид номенклатуры: "Товар", "Услуга". Заполнять единицы измерения, ставки НДС
    📄 Заказ клиента Продажи → Заказы клиентов 🎯 Основа для работы! Статусы: "Согласование", "К отгрузке", "Закрыт"
    📄 Документы продажи (все) Продажи → Реализация товаров и услуг 📦 Оформляется на складе ГП. Проводки по управленческому и регламентированному учету

    2️⃣ Главный инженер

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Основные средства Внеоборотные активы → Основные средства → Документы по ОС 🏭 Оборудование, транспорт, здания. ⚠️ В КА учет ОС ведется параллельно с регламентированным учетом
    📘 Виды работ Производство → Трудозатраты → Виды работ 🔨 Список технологических операций с указанием расценок и трудозатрат
    📄 Плановые ремонты (График ППР) Производство → Настройки и справочники по производству → Графики работы 📅 Планирование ТО и ремонтов оборудования
    📄 Заказ-наряд на ремонт Производство → Заказы перерабочикам 🔧 Учет фактических затрат на ремонт (запчасти, время)

    3️⃣ 📊 Инженер ПДО (Планово-диспетчерский отдел)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Ресурсные спецификации Производство → Ресурсные Спецификации 🧩 Конструкторский и технологический состав. Тип: "Сборочная единица"
    📘 Статьи калькуляции Производство → Статьи калькуляции ⏱️ Последовательность операций, нормы времени, рабочие центры
    📄 Заказы переработчикам Производство → Заказы пеереработчикам 🚀 Запуск изделия. Заполнять: номенклатура, количество, плановая дата, спецификация
    📄 Настройки и справочники по производству -> Графики работы Производство → Настройки и справочники по производству → Графики работы 📅 Распределение заказов по сменам и рабочим центрам

    4️⃣ ️ Технолог

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Номенклатура НСИ и Администрирование → НСИ → Номенклатура 🏭 Создание карточек ГП и полуфабрикатов. Тип: "Товар", вид: "Продукция"
    📘 Спецификации номенклатуры Производство → Ресурсные Спецификации 📦 Материальный состав (что входит). ⚠️ В КА есть понятие "Спецификация-этап"
    📘 Технологические карты Оформляется через номенклатуру и специфкации 🛠️ Детальное описание операций с привязкой к рабочим центрам и инструменту
    📄 Этапы производства Производство → Этапы производства 🔄 Используется для маршрутного учета при длительном цикле

    5️⃣ Конструктор

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Номенклатура НСИ и Администрирование → НСИ → Номенклатура 📝 Создание позиций деталей, сборочных единиц
    📘 Спецификации номенклатуры Производство → Ресурсные Спецификации 🏗️ Конструкторский состав (вложенность до 5-10 уровней)
    📘 Характеристики номенклатуры НСИ и Администрирование → НСИ → Номенклатура (характеристики) 📏 Размеры, цвета, параметры (если требуется)
    📎 Вложения в файлы Кнопка "📎 Скрепка" в карточке номенклатуры 📁 Прикрепление чертежей, 3D-моделей, ТУ

    6️⃣ Снабженец (Отдел закупок)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Контрагенты (Поставщики) НСИ и Администрирование → НСИ → Партнеры 🤝 Категория "Поставщик", договоры, условия оплаты
    📄 Заказ поставщику Закупки → Заказы поставщикам 📋 Формируется на основании потребности (по дефициту или заказу клиента)
    📄 Поступление товаров и услуг Закупки → Документы закупки (все) 📥 Оприходование ТМЦ на склад. ⚠️ В КА два вида: управленческое и регламентированное поступление
    📊 Потребности в закупках Закупки → Состояние обеспечения заказов 🔍 Анализ того, что нужно закупить для обеспечения производства и заказов

    7️⃣ Кладовщик (Материалы и НЗП)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Склады (и ячейки) НСИ и Администрирование → НСИ → Склады 🏢 Типы складов: "Материальный", "Производственный (НЗП)"
    📄 Оприходование товаров Склад и доставка → Поступления товаров (или Акт оприходования) ✅ Приемка материалов от поставщика
    📄 Перемещение товаров Склад и доставка → Перемещения товаров 🚚 Выдача материалов в цех (со склада материалов на склад НЗП)
    📄 Инвентаризация товаров Склад и доставка → Пересчеты товаров 🔢 Сверка фактических остатков материалов с учетными
    📊 Остатки товаров Склад и достпвка → Отчеты по складу → Остатки и доступность товаров 📋 Ведомость по складам материалов

    8️⃣ Начальник производства (Мастер цеха)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Рабочие центры Производство → Настройки и справочники по производству → Производственные подразделения 🔧 Станки, участки (Фрезерный, Сварочный)
    📄 Производственный заказ Производство → Заказы переработчикам 🎯 Задание для цеха. ⚠️ В КА это основной документ для планирования!
    📄 Отчет производства за смену Производство → Отчеты по производству ✅ Фиксация выработки, брака, времени работы сотрудников и оборудования
    📄 Требование-накладная Производство → Передача в производство 📤 Списание материалов в НЗП (на основании спецификации)

    9️⃣ Кладовщик (Готовая продукция)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Склады (ГП) Склад и доставка → Приемка 📦 Тип склада: "Готовая продукция"
    📄 Выпуск продукции Производство → Документы производства (все) 📥 Оприходование ГП из цеха на склад. Основание — Отчет производства за смену
    📄 Реализация товаров и услуг Продажи → Документы продажи (все) 🚚 Отгрузка со склада ГП клиенту
    📄 Комплектация (Сборка) Склад и достпвка → Создать → Заказ на сборку (разборку) 📦 Сборка комплектов, упаковка, разукомплектация
    📊 Ведомость по товарам на складах Склад и доствка → Отчеты по складу → Ведомость по товарам на складах 📈 Анализ остатков ГП

    🔟 ✅ Инженер ОТК

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📄 Отчет производства за смену Производство → Отчеты по производству ⚠️ В документе заполняется колонка "Брак", указывается код брака
    📄 Акт о браке НСИ и администрирование → Номенклатура → Качество товаров ❌ Оформление исправимого/неисправимого брака. Заполняется виновник, статья затрат
    📄 Приемка товаров и услуг (Качество) Производство → Отчеты по производству → Контроль поставки продукции переработчиком ↩️ Оформление возврата поставщику при входном контроле
    📘 Сертификаты качества Зависит от места обнаружения (производство, склад, покупатель) и типа брака (исправимый/неисправимый) 📄 Привязка сертификатов к партиям товара

    1️⃣1️⃣ Отдел реализации (Склад ГП + Отгрузка)

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📄 Заказ клиента Продажи → Заказы клиентов 🔒 Резервирование ГП на складе под конкретный заказ
    📄 Счет на оплату покупателю Продажи → Счет на оплату покупателю 💰 Выставляется клиенту
    📄 Реализация товаров и услуг Продажи → Документы продажи (все) 📤 Оформление отгрузки, печать ТОРГ-12
    📊 Заказы клиентов к отгрузке Продажи → Электронные перевозочные документы 📋 Список заказов, ожидающих отгрузки

    📘 1С:Комплексная автоматизация (КА) — Экономист

    💡 В КА функционал экономиста значительно шире за счет разделения управленческого и регламентированного учета, наличия полноценного блока бюджетирования и более развитого учета затрат.

    Тип Название объекта Путь в интерфейсе 💡 Основные поля / Примечание
    📘 Статьи затрат Финансовый результат и контроллинг → Доходы и расходы, финансовый результат 📋 Управленческие статьи. ⚠️ Есть иерархия и правила распределения
    📘 Статьи движения ДС Казначейство → Отчеты по казначейству → Движение денежных средств 💰 Ключевой справочник для Платежного календаря и анализа ДДС
    📘 Номенклатурные группы НСИ и Администрирование → НСИ → Номенклатура 📦 Для аналитики рентабельности по группам продукции
    📘 Подразделения (ЦФО) НСИ и Администрирование → НСИ → Структура предприятия 🎯 В КА подразделения могут быть Центрами финансовой ответственности
    📘 Проекты НСИ и Администрирование → НСИ → Проекты 📁 Дополнительный разрез аналитики (заказы, гранты, направления)
    📄 Заявка на расходование ДС Казначейство → Заявки на расходование ДС 📝 Внутренний документ-основание. ⚠️ Утверждается экономистом/финдиректором
    📄 Платежный календарь Казначейство → Платежный календарь 📅 Планирование платежей с учетом лимитов и приоритетности
    📄 Платежное поручение Казначейство → Распоряженние на перемещение денежных средств 📤 Создаются на основании утвержденных заявок
    📄 Выписка банка (загрузка) Казначейство → Банк → Безналичные платежи 🏦 Загрузка фактических движений ДС, автоматическое отражение в учете
    📄 Распределение доходов и расходов Финансовый результат и контроллинг → Доходы и расходы, финансовый результат 🔄 Закрытие месяца, распределение накладных расходов по ЦФО, проектам
    📄 Расчет себестоимости Финансовый результат и контроллинг → Отчеты по финансовому результату → Закрытие месяца → Сверка остатков оперативного учета и себестоимости ⚙️ Регламентная операция для расчета фактической себестоимости выпуска
    📄 Корректировка долга Финансовый результат и контроллинг → Учет доходов и расходов → Начисление и списание резервов по сомнительным долгам ⚖️ Для списания задолженности, взаимозачетов
    📄 Бюджет (Сценарий) Бюджетирование и планирование → Бюджеты → Экземпляры бюджетов 📊 Плановые показатели по доходам, расходам, ДДС, балансу. Поддержка сценариев: План, Факт, Прогноз
    ⚙️ Бюджетное правило Бюджетирование и планирование → Настройки и справочники по бюджетированию и планированию 🤖 Правила автоматического расчета бюджетов на основе данных прошлых периодов
    📊 Финансовый результат (упр.) Финансовый результат и контроллинг → Отчеты по финансовому результату 🎯 Анализ прибыли и рентабельности в разрезе групп, ЦФО, проектов
    📊 ОСВ (управленческая) Казначейство → Отчеты по казначейству 📈 Управленческие обороты по счетам
    📊 Задолженность контрагентов Финансовый результат и контроллинг → Управленческий баланс → Статьи активов и пассивов → Дебиторская задолженность ⚠️ Дебиторка и кредиторка в управленческом учете
    📊 Отчет о движении ДС Регламентированный учет → Стандартные бухгалтерские отчеты → Оборотно-сальдовая ведомость 💵 План-фактный анализ платежей по статьям ДС

    ➕ Дополнительные объекты (регламентированный учет)

    Тип Название объекта Путь в интерфейсе 💡 Примечание
    📄 Операция (бухгалтерская) Регламентированный учет → Операции ✍️ Для корректировок, не проходящих через стандартные документы
    📊 Бухгалтерский баланс (ф. №1) Регламентированный учет → Отчеты → Бухгалтерский баланс 📑 Регламентированная отчетность для налоговой
    📊 ОСВ (регламентированный учет) Регламентированный учет → Отчеты → ОСВ 🔍 Для сверки с налоговой инспекцией и аудита

    🎯 Рекомендации для экономиста

    ✅ Универсальные правила:

    • 🔍 Проверяйте аналитику при вводе документов (ЦФО, проекты, номенклатурные группы)
    • 💾 Регулярно проводите закрытие месяца для корректного расчета себестоимости
    • 📊 Используйте план-фактный анализ для контроля отклонений
    • 🤝 Согласовывайте заявки на ДС с руководителями подразделений

    🔥 Важные моменты в КА:

    1. Разделяйте управленческий и регламентированный учет — не путайте документы!
    2. Используйте ЦФО для детализации ответственности по затратам
    3. Настраивайте бюджетные правила — это экономит время при планировании
    4. Мониторьте лимиты в Платежном календаре для предотвращения кассовых разрывов

    ⚠️ Частые ошибки:

    Ошибка Последствие Как избежать
    ❌ Не заполнены аналитики в статьях затрат Некорректная себестоимость ✅ Проверять обязательные разрезы аналитики
    ❌ Пропущено распределение накладных расходов Заниженная/завышенная прибыль ✅ Запускать регламентные операции закрытия месяца
    ❌ Не утверждены заявки на ДС Блокировка платежей в календаре ✅ Своевременно согласовывать заявки
